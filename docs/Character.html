Class: Character

Character(nameopt, typeopt, progressionPointsopt)

Represents a character with progression tracking across different life stages.

Constructor

new Character(nameopt, typeopt, progressionPointsopt)

Creates an instance of a Character. If no name and type are provided, it loads the first available character from storage.
Parameters:
Name Type Attributes Default Description
name string <optional>
 null The name of the character.
type string <optional>
 null The type of the character (Monkey, Dragon, TBD).
progressionPoints number <optional>
 0 Initial progression points (default is 0).
